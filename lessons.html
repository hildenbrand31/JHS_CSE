The web contains a seemingly near infinite number of resources for learning web and app development. I only know a few but will share the good ones here. This is a collection of useful links for reference and learning html, CSS, JavaScript, PHP, and MySQL.

Learning Sites and Tutorials
Link Description
w3schools A great site with tutorials and reference materials for web page and web app development.
Codecademy The best coding instruction site on the web. This is where I started to learn anything at all about code.
tuts+ Web Development Another good site with a lot of quality tutorials on web design. Other categories as well.

Stand-Alone Local Web Stacks
Name Description
MAMP Initially developed for the Mac; a Windows version promised to come soon. I use it and like it a lot.
WampServer WampServer is a Windows web development environment. I use it in the classroom and it works well enough.
xampp XAMPP is a third alternative. It runs in Windows, Mac OS X, and Linux. Choice is good. And this comes highly recommended.
